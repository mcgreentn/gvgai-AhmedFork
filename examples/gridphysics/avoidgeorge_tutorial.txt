The game is avoidgeorge.
You are the ShootAvatar.
To move as the ShootAvatar, use the arrow keys to turn and move.


This is how you win:
If time reaches 1000 seconds, then you will win.

This is how you lose:
If avatar (citizen) and george (Chaser) collide, then the avatar (citizen) sprite will be destroyed.
If there are no more avatar (citizen) sprites then you will lose.

This is how you get points:
If the annoyed (human) and the cigarette (Flicker) collide, then you will gain 1 point.
